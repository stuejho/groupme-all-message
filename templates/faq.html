{% block title %}FAQ{% endblock %}

  • You can only send group mentions to groups where you are an admin or owner.

  • First, you authenticate using GroupMe's OAuth portal. Then, you are redirected back to this site with an access token. This token is briefly processed by the site server and then locally stored on your browser. The stored token is used whenever you send a message. This site does not see your outgoing messages.

  • The maximum message size is 990 characters.

  • Support for sending images/attachments is not supported.
